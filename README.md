Cyber Security Task 1 – Web Application Vulnerability Assessment

Tools Used

Kali Linux

OWASP ZAP

Web Browser

Description

This project demonstrates performing a web application vulnerability assessment using ethical hacking techniques.
The objective was to identify common security vulnerabilities in a test web application and analyze their impact based on OWASP Top 10 guidelines.

Vulnerabilities Identified

SQL Injection vulnerability

Cross-Site Scripting (XSS) vulnerability

Broken Authentication issue

Details of Findings

SQL Injection allowed manipulation of backend database queries

XSS vulnerability enabled execution of malicious scripts in the user’s browser

Broken Authentication exposed weaknesses in login and session handling

Outcome

Gained hands-on experience in:

Web application security testing

Identifying and analyzing OWASP Top 10 vulnerabilities

Using OWASP ZAP for vulnerability scanning

Creating professional security assessment reports
